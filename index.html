Cellular Automata Explorer

About Cellular Automata Explorer

This interactive app provides a UI for exploring cellular automata (CA) by exposing granular controls for most of the important parameters including survival/birth counts, neighborhood type, range, and more.

Currently this app only supports so-called "Life-like" rules, including Larger than Life and Generations. Square cells in a regular grid with discrete, deterministic states

Keyboard

  • Space - pause/play
  • r - restart
  • u - show/hide the UI
  • h - show this "about" panel

Mouse

  • Click and drag to set all cells within the brush radius to 0.
  • Use the mouse wheel to make the brush size bigger or smaller.

Accessibility

  • All UI controls can be reached and operated by keyboard.
  • To paint on the canvas, first set focus onto the canvas, then use the following keys to interact with it:
    • Hold down Space to touch the brush to the canvas. Release to lift it.
    • Use your arrow keys to move the brush around.
    • Hold down Shift while moving to go faster. Release it to return to normal speed.
